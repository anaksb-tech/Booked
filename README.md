# Booked

## Visão Geral do Projeto

### O sistema Booked servirá como um ecossistema digital para amantes da leitura, unindo um marketplace de e-books independentes a um clube de trocas de livros físicos baseado em geolocalização. O fator motivacional para o desenvolvimento do sistema é a fomentação da circularidade de livros físicos, assim promovendo a sustentabilidade literária e tirando livros que ficariam parados nas estantes. Além disso, a plataforma oferece uma vitrine direta para e-books, para conectar autores independentes aos leitores, assim os autores ganham visibilidade estratégica e o autor detém o controle total sobre sua obra.
### No sistema, cada usuário poderá criar um perfil personalizado, gerenciar sua estante virtual (livros físicos que possui e deseja trocar) e sua biblioteca digital (que conta com ebooks adquiridos ou publicados). Autores independentes podem publicar suas próprias obras, definindo valores e descrições para venda direta na plataforma. Para os livros físicos, o sistema utiliza um algoritmo de matchmaking que identifica automaticamente quando o "Usuário A" possui um livro desejado pelo "Usuário B", assim o sugerindo a troca. O algoritmo de matchmaking será baseado no cruzamento automatizado entre as listas de desejos e o histórico de troca dos usuários, utilizando categorias literárias e autores de preferência como critérios de relevância para sugerir combinações de troca mútua.
### As trocas físicas são facilitadas por filtros de proximidade (região ou instituição de ensino), permitindo que usuários negociem e finalizem transações presencialmente ou via envio. Cada negociação possui uma automação de status, transitando entre estados como "Disponível", "Em negociação", "Enviado" e "Recebido". Para garantir a segurança e o engajamento, o Booked conta com um Rank de Confiabilidade gamificado: usuários acumulam pontos, medalhas e reputação positiva ao completarem trocas bem-sucedidas e avaliarem obras de forma construtiva. O sistema gerencia o fluxo de troca através de validação mútua: em entregas presenciais, utiliza-se um código de confirmação entre os usuários; em envios remotos, utiliza-se o registro de rastreamento. A pontuação no Rank de Confiabilidade é vinculada estritamente à conclusão dessas etapas, servindo como selo de segurança para futuras negociações.
